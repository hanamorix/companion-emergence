# v0.0.11-alpha.5 тАФ 2026-05-14

## Public release state

- Git tag: `v0.0.11-alpha.5`
- Release URL: <https://github.com/hanamorix/companion-emergence/releases/tag/v0.0.11-alpha.5>
- Package version: `0.0.11`

## What changed

This release fixes the Windows WebView2 bridge fetch regression and adds
a past-image gallery panel plus auto-update infrastructure.

### Windows WebView2 fetch fix

Root cause: Tauri 2 serves Windows frontends from `https://tauri.localhost`
(public address space) while the bridge listens on `http://127.0.0.1`
(private address space). Chromium's Private Network Access blocked all
fetches before any request left the browser. Fix: `useHttpsScheme: false`
in the Tauri window config. No change to macOS or dev mode.

### Past-image gallery (new)

New Gallery tab (тЧл) in the left panel shows every image shared across all
past conversations as a thumbnail grid. Click any thumbnail for a full-size
lightbox (Escape or click backdrop to close). Works cross-platform with
no native code.

### Auto-update support (new)

The app can now check for, download, and install updates from GitHub
Releases. Find it in the Connection panel under the new "Updates" section.
Built on `tauri-plugin-updater` 2.x with cryptographic signatures. macOS
uses DMG, Windows uses MSI, Linux uses AppImage.

### Infrastructure

- Public sync filter-repo recovery: post-filter step restores files dropped
  by merge simplification, with personal-marker filtering applied.
- CI now generates cryptographically signed `latest.json` for the updater.
- Updater manifest served from `latest-release` branch for pre-release compat.

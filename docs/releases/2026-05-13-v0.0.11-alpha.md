This release packages the initiate-physiology arc:

  voice-edit proposals, draft-space fallback, UI banners, and operator CLI views.

  Haiku-backed research topic-overlap scoring.

  the bundled brain instead of looking for a Unix-shaped `bin/nell` entry point.
